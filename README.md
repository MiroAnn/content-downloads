# Content downloads

Публичная страница для скачивания готовых материалов проекта.

## Как добавлять новые карусели

- Каждую карусель сохранять в новой уникальной папке внутри `carousels/`.
- Не заменять файлы уже опубликованных каруселей.
- Новый блок карусели добавлять в `index.html` перед предыдущими блоками.
- Для каждого слайда использовать ссылку с атрибутом `download`, чтобы PNG скачивался по клику.
- Публиковать только отдельные PNG; ZIP-файлы не создавать и не добавлять на страницу.
